# pwr-server
My web server for controlling CoAP-enabled power relay.
Written in Rust, using
  * Actix-web framework
  * Askama template engine
  * CoAP client library

### References

  * https://actix.rs/
  * https://crates.io/crates/actix-web
  * https://github.com/djc/askama
  * https://docs.rs/crate/coap/
